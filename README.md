This project contains two main folders:

1. **backend_frontend**:  
    - `frontend_wanlp`: A React-based frontend application.  
    - `hate_speech_api`: A FastAPI-based backend service.

2. **model**:  
    - Contains datasets and two models: the base model and the LoRA adapter.

3. **presentation**:
    - A PDF file for the presentation
